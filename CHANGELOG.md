## [2.2.2] - 2025-05-23

### 🔧 Critical Bug Fixes (תיקונים קריטיים)
- **🔧 תיקון טעינת DOCX**: פתרון שגיאות "Failed to load DOCX from CDN" עם מערכת טעינה משופרת
- **📁 אבטחת סיומות קבצים**: וידוא שכל הקבצים המיוצאים מקבלים סיומת נכונה (.xlsx, .csv, .docx)
- **⚡ שדרוג מערכת CDN**: מערכת טעינה חכמה עם timeout של 15 שניות וטיפול משופר בשגיאות
- **🛡️ אתחול מוגן**: הוספת בדיקת dependency מלאה ומנגנוני fallback
- **🔍 שיפור לוגים**: מערכת logging משופרת לאבחון בעיות
- **🎯 אפשרויות דינמיות**: עדכון ממשק המשתמש בהתאם לפורמט הנבחר

### ✅ Excel Export Improvements (שיפורי ייצוא Excel)
- **📊 וידוא סיומת .xlsx**: תיקון מקרים שבהם קבצי Excel חסרו סיומת נכונה
- **🔄 fallback משופר**: מנגנון הורדה כפול (ישיר + blob) לתאימות מקסימלית
- **📈 רוחב עמודות טוב יותר**: חישוב אוטומטי של רוחב עמודות לתצוגה אופטימלית
- **🔤 תמיכה משופרת ב-RTL**: טיפול משופר בטקסט מימין לשמאל בתאי Excel

### 🎨 User Experience Enhancements (שיפורי חוויית משתמש)
- **📱 התרעות מידע**: הודעות זמניות על הצלחה/מידע לפידבק טוב יותר
- **🎨 אפשרויות דינמיות**: הצגה/הסתרה של אפשרויות בהתאם לבחירת הפורמט
- **🔧 הודעות שגיאה משופרות**: הודעות שגיאה אינפורמטיביות יותר בעברית
- **⚡ טעינה מהירה יותר**: סדר טעינת סקריפטים מותאם ואתחול משופר

### 🐛 Bug Fixes (תיקון באגים)
- **🎯 תיקון שגיאת favicon 404**: הוספת favicon.ico נכון
- **⚙️ תיקון שגיאות JavaScript**: פתרון שגיאות syntax בטעינת מודולים
- **📁 תיקון בעיות סיומת קבצים**: שיפור בדיקת סיומות קבצים
- **🔧 שיפור טיפול בכשלי CDN**: טיפול משופר בכשלי טעינה מ-CDN

### 🛠️ Technical Improvements (שיפורים טכניים)
- **📡 מערכת CDN חכמה**: מעבר אוטומטי בין unpkg.com → cdnjs.com → קובץ מקומי
- **🔍 בדיקות dependency**: בדיקת זמינות מחלקות JavaScript לפני שימוש
- **⚡ Timeout mechanisms**: מנגנון timeout של 15 שניות לטעינת ספריות
- **📊 Enhanced console logging**: לוגים מפורטים יותר לאבחון בעיות
- **🎯 Format-specific validation**: בדיקות ספציפיות לכל פורמט יצוא

### 📚 Documentation Updates (עדכוני תיעוד)
- **📖 README מעודכן**: הוספת מידע על התיקונים והשדרוגים החדשים
- **🔧 מדריך troubleshooting**: מדריך מפורט לפתרון בעיות נפוצות
- **📋 CHANGELOG מעודכן**: תיעוד מפורט של כל השינויים
- **🧪 test-docx.html**: כלי בדיקה עצמאי לוודא תקינות ספריית DOCX

## [1.0.0] - 2025-05-22 (גרסה ראשונה)

- **👁️ OCR מתקדם** עם Tesseract.js לזיהוי טקסט עברי
- **🔄 עיבוד קבוצתי** של מספר קבצי PDF
- **👀 תצוגה מקדימה** של נתונים לפני הורדה
- **✅ אימות נתונים אוטומטי** עם דוח מפורט
- **📱 עיצוב responsive** לכל המכשירים
- **🎨 ממשק משתמש מודרני** עם Bootstrap 5
- **🌐 תמיכה מלאה ב-RTL** ועברית
- **🖱️ Drag & Drop** להעלאת קבצים
- **📊 התקדמות בזמן אמת** עם אינדיקטורים

### 🔧 Technical Features (תכונות טכניות)
